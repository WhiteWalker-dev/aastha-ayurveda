# Aastha Ayurveda Website

A premium, highly responsive, and SEO-optimized website for **Aastha Ayurveda** clinic and training academy.

This project represents over two decades of clinical healing practice under Dr. Sandeep Madaan, featuring comprehensive clinical services, training program modules, and an interactive contact interface.

## Tech Stack
*   **Core**: HTML5, Vanilla CSS3, Vanilla JavaScript
*   **Design Tokens**: Custom HSL/Hex theme based on Ayurvedic Deep Green (`#0F766E`) and Warm Gold (`#D97706`).
*   **Typography**: *Playfair Display* (Serif) for headings and *Plus Jakarta Sans* (Sans-serif) for body copy.
*   **Features**: Dynamic scroll fade-in reveals, sticky navigation bar, mobile burger drawer, accordion FAQs, social media links integration, and interactive Google Maps embed.

## Site Map
1.  **Home** (`index.html`): Overview of clinical therapies, doctor profile, trust facts, reviews, and general FAQs.
2.  **Treatments** (`treatments.html`): Detailed diagnostic cards and treatments for Digestive, Joint Pain, Migraine, Skin, Panchakarma, and Women's Health.
3.  **Training Academy** (`training.html`): Directory of certified courses (Basic, Advanced, Fellowship) and student application form.
4.  **About Us** (`about.html`): Deep dive into the history, mission, vision, and Dr. Sandeep Madaan's professional CV timeline.
5.  **Contact** (`contact.html`): Quick call details, inquiry form (with auto-prefill logic), and Google Maps location.

## Getting Started

To run the site locally with live reloading:

1.  Make sure you have [Node.js](https://nodejs.org) installed.
2.  Open your terminal in the project directory and run:
    ```bash
    npm run dev
    ```
    This script will invoke `browser-sync` automatically without needing to install global dependencies.
